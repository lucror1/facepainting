Cube Controls

  • Click + drag: rotate cube
  • Click + shift + drag: move cube

Face Painting Controls

  • Click once: select face
  • Click after selected: paint on face
  • Escape key: deselect current face
  • Click outside of a face: deselect current face

Painting Settings

  • Line thickness: the thickness of the line
  • Color: the color of the line to be drawn
  • Background color:
    • If a face is selected: set the background color for that face
    • If no face is selected: set the background color for all faces
  • Update: update the cube with the painted faces
  • Reset:
    • If a face is selected: reset the selected face
    • If no face is selected: reset all faces and settings
